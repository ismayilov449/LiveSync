## Recording tips

- **Resolution:** 1280×720 or 1920×1080
- **Length:** 15–30 seconds per GIF
- **Size:** Keep under 10 MB for GitHub (use ScreenToGif optimizer)
- **Focus:** Show the item appearing in the *other* tab without refresh
